# Hosts管理工具!
主页：[https://github.com/pglin66/HBuilder-plin-hosts](https://github.com/pglin66/HBuilder-plin-hosts)

这是一个用于快速切换 hosts 文件的小程序，基于 [HBuilder X,nodejs] 开发，使用了  [CodeMirror](http://codemirror.net/) 框架/库。

## 功能特性
 - 快速切换 hosts
 - hosts 文件语法高亮
 - 在线 hosts 方案
 - HBuilder X里快速切换
